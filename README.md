# Electronics Project

This is a KiCad electronics project for Air Quality Project.

## Project Structure

- `*.sch` → Schematic files
- `*.kicad_pcb` → PCB layout files
- `*.lib` → Component libraries
- `.gitignore` → Files to ignore (build outputs, backups, temporary files)

## Getting Started

1. Install [KiCad](https://kicad.org/)
2. Open the `.pro` or `.kicad_pcb` files to view/edit the project
3. Generate Gerber files for manufacturing if needed
